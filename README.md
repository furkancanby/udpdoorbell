# udpdoorbell
UDP Pinger Between Server and Client

Usage:

Server --> py udppinger.py -s "Port"
Client --> py udppinger.py -c "IP Adress" "Port"

Example:

py udppinger.py -s 4444
py udppinger.py -c 192.168.1.24 4444
